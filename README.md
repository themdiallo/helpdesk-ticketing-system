# Helpdesk Ticketing System

A real-world IT support ticketing system project focused on databases, backend development, and full-stack engineering.

---

## Project Goals

This project is designed to simulate a professional helpdesk environment where users can:

- Submit support tickets
- Track ticket status
- Categorize technical issues
- Manage users and support staff
- Improve IT support workflows
